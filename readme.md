## 2. 프로젝트 목표

**데이터 분석 결과로 도출되는 인사이트와 웹서비스의 해결과제에 대한 논의 (50자 이상)**
  - 프로젝트 아이디어 동기
  - 문제를 해결하기 위한 특정 질문 명시
  - 데이터를 통해 탐색하려는 문제를 구체적으로 작성
  - 위드 코로나 시대로 변하게 되면서 여행 수요가 다시 증가하고 있는 상황에서 각 나라별로 다른 여행 조치를 시행하고 있음, 이에 따라 여행자들은 각 나라별로
